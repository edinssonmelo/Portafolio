# Carpeta de Cotizaciones

Coloca aquí el archivo PDF de la cotización.

## Instrucciones

1. Coloca tu archivo PDF en esta carpeta con el nombre: `sistema-barber.pdf`
2. El archivo será accesible en la ruta: `/cotizacion/sistema-barber`
3. El PDF se mostrará automáticamente en la página de cotización

## Ubicación del archivo

El archivo debe estar en: `public/cotizaciones/sistema-barber.pdf`

## Nota

Asegúrate de que el nombre del archivo sea exactamente `sistema-barber.pdf` (en minúsculas) para que funcione correctamente con la ruta configurada.
